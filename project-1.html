Push-Up Video Classification

Sistema inteligente de corrección de técnica deportiva utilizando Deep Learning y redes de grafos espacio-temporales (ST-GCN).

Sobre el Proyecto

Este proyecto explora la intersección entre el deporte y la inteligencia artificial avanzada. El objetivo principal fue desarrollar un sistema capaz de analizar vídeos de usuarios realizando flexiones (push-ups) y determinar automáticamente si la técnica empleada es correcta o incorrecta.

La solución no se limita a un simple análisis frame a frame, sino que entiende la dinámica temporal del movimiento. Utilizando MediaPipe para la extracción precisa de puntos clave del cuerpo (pose estimation) y redes Bi-LSTM junto con ST-GCN, el modelo logra capturar dependencias espaciales y temporales complejas.

Características Clave

  • check_circle Extracción de pose corporal en tiempo real con más de 30 landmarks.
  • check_circle Modelado de relaciones espaciales entre articulaciones y temporales entre frames (ST-GCN).
  • check_circle Alta precisión en la distinción de errores técnicos sutiles.
  • check_circle Pipeline optimizado para inferencia en CPU utilizando ONNX (opcional).

Tecnologías

Python PyTorch MediaPipe ST-GCN Bi-LSTM
