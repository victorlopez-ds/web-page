Customer Churn Prediction

Predicción de abandono de clientes a escala masiva utilizando computación distribuida con Apache Spark.

Sobre el Proyecto

En la era del marketing digital, retener a un cliente es mucho más rentable que adquirir uno nuevo. Este proyecto aborda el problema de la predicción de bajas (churn) simulando un entorno de producción con grandes volúmenes de datos donde las herramientas tradicionales como Pandas no serían suficientes.

La solución se basa en Apache Spark (PySpark), permitiendo el procesamiento distribuido de millones de registros. Se implementó un flujo completo de Data Science (Pipeline), desde la limpieza y transformación de datos hasta el entrenamiento de modelos de clasificación y su evaluación.

Características Técnicas

  • check_circle Procesamiento Distribuido: Uso de RDDs y DataFrames para manipulación escalable.
  • check_circle Feature Engineering: Transformación de variables categóricas y numéricas con `VectorAssembler` y `StringIndexer`.
  • check_circle Modelado Comparativo: Evaluación de Logistic Regression, Decision Trees y Gradient-Boosted Trees (GBT).
  • check_circle Optimización de hiperparámetros y validación cruzada.

Tecnologías

Python PySpark MLlib Jupyter Big Data
